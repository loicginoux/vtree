--- layout: api title: Checkbox API permalink: /api/plugins/checkbox js_object: checkbox ---

This plugin add a checkbox to a node. You can parameter the initial state of the checkbox and disable it. there is also a 'checkbox.node' event added to the container. You can choose to display or not the checkbox in order to use just a bold state. You can also change the behaviour when clicking a checkbox, if it should click all his parents or all his children.
