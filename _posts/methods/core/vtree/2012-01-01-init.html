--- post_type: method js_object: vtree function: init title: init(settings, className) param1_name: settings param1_description: settings contains an array of plugin names. these plugins will be added to the Object represented by 'className' param2_name: className param2_description: className is a string that represents an Object. It can be either 'tree', 'node' or 'nodeStore'. --- this is used internally to attach all plugins to the object 'this'. Every core object have in their constructor something like: {% highlight js%} Vtree.init.apply(this, [settings, ""]) {% endhighlight %} where classname is node, tree or nodeStore. It attaches all core functions/variables and all plugin functions/variable to the className object.
