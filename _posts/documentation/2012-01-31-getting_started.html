First steps

This is what you need to do to get started:

  • Download and uncompress the archive.
  • install the files under the dist directory into your own application. The minimum file you need are vtree.js and vtree.css. Then If you need some icons, install the images too.
  • This project is based on jQuery and json2. If you haven't installed this files yet in your application, you can find them under the lib directory.
  • Follow this basic example to get your first tree.
  • You will find different examples in the archive as well if you need more help.
  • If there is anything wrong, just open an issue on github
