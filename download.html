What's included?

You are going to download a zip file containing:

  • vtree.js - containing the core functions + all plugins
  • vtree.min.js - the minified version
  • vtree.css - a css file containing default styling
  • vtree.min.css - the minified version

What's not included?

This libraries are necessary to make it work, they are not included because they are often already parts of web projects:

  • jquery - I guess you didn't know this one before!
  • json2 - to parse and stringify JSON objects
Download Vtree.zip

More customization?

If you need more control about what you want or don't want to include, I advise you to directly clone into the Vtree github repository and use the makefile to generate everything nicely.
